# 固定资产管理系统 python317

## 项目概述

固定资产管理系统是一款基于Python语言和Vue前端框架开发的管理系统。主要用于企业或机构的资产管理工作，实现资产的全面监控与高效管理。

## 技术栈

- 前端：Vue
- 后端：Python FastAPI

## 功能模块

### 资产管理

- 资产入库
- 资产出库
- 资产领用
- 资产借用
- 资产归还
- 资产维修
- 资产报废

### 统计分析

- 首页数据统计
- 资产清单
- 资产分类汇总
- 闲置资产统计

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 后端：Python 3.8+
- 前端：现代浏览器

## 目录结构

```
├── backend                # 后端代码
│   ├── app                # FastAPI 应用代码
│   ├── databases          # 数据库相关
│   ├── dependencies       # 依赖注入
│   ├── models             # 数据模型
│   └── schemas            # Pydantic 模型
└── frontend               # 前端代码
    ├── assets             # 静态资源
    ├── components         # Vue 组件
    ├── views              # 页面视图
    └── router.js          # 路由配置
```

## 部署步骤

1. 准备Python运行环境及依赖库
2. 安装FastAPI框架
3. 配置数据库
4. 启动后端服务
5. 构建Vue前端项目
6. 部署前端项目至Web服务器
